# Checklist Pre-PR

## Escopo e intenção
- [ ] Escopo mínimo e explícito
- [ ] Sem refatoração não relacionada

## Qualidade
- [ ] Testes adicionados/atualizados para o comportamento alterado
- [ ] Lint/format/checks executados (ou justificativa explícita)
- [ ] Risco classificado e documentado

## Contratos e compatibilidade
- [ ] Compatibilidade de API/evento/schema avaliada
- [ ] Breaking changes explicitamente aprovados

## Segurança e compliance
- [ ] Sem segredos hardcoded
- [ ] Sem dados sensíveis em logs/erros
- [ ] Impactos de autenticação/autorização revisados

## Operação
- [ ] Impacto de observabilidade avaliado
- [ ] Plano de rollback documentado
- [ ] Impacto de CI/CD avaliado

## Documentação
- [ ] README/docs/ADR atualizados quando necessário
- [ ] Descrição de PR com evidências e validação
